私の好きなHTML要素

今回は私が好きなHTML要素を3つほど紹介していきたいと思います。

  • 紹介するHTML要素
    1. a要素
    2. footer要素
    3. div要素
  • 参考文献

a要素

a要素はhref属性を用いてリンクを埋め込み、別のWebページやファイル、同一ページの指定の場所等のハイパーリンクを作成します。 a要素の内容はリンクであることが多いです。

a要素の好きなところは、p要素とは異なり、文章だけでなくリンクの埋め込みが可能であるところです。

  • 使用例
                
                <a href = "https://html.spec.whatwg.org/multipage/#toc-semantics">4 The elements of HTML</a>
  • 結果
4 The elements of HTML

footer要素

footer要素は、ページ下部に情報を埋め込み表示させる要素です。footer要素に埋め込む情報として、そのページに関する著作情報や関連記事へのリンクなどが あります。

私が、footer要素のことが好きな理由は、べージの下部に概要などの情報を固定することが出来る事です。

  • 使用例
                
                <footer>2021-11-07</br>Web Programming</footer>
  • 結果
2021-11-07
Web Programming

div要素

div要素は、単体ではあまり意味を持たない要素です。div要素はブロックとして複数の要素を囲い、まとめて処理を設定し、行うことが出来ます。 div要素を用いると、CSSでのスタイリングを、classを定義した時と似たようにまとめて反映させることが出来ます。

私がdiv要素を好きな理由は、div要素が特定の意味を持たないので、class属性を用いてある程度自由に要素の役割を決められるからです。

  • 使用例
                
                <div class = "result">
                    <ul>
                        <li>example1</li>
                        <li>example2</li>
                        <li>example3</li>
                    </ul>
                </div>
  • 結果
  • example1
  • example2
  • example3
